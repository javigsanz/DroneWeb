# WebJ2Drones Landing (Angular 20)

Landing page premium para empresa de grabacion aerea con drones.

## Stack
- Angular 20
- TypeScript
- SCSS
- HTML semantico
- Animaciones nativas con IntersectionObserver y requestAnimationFrame

## Ejecutar
1. Instala Node.js 20+
2. Instala dependencias:
   npm install
3. Levanta entorno local:
   npm start
4. Build produccion:
   npm run build

## Estructura principal
- src/app/sections: secciones independientes de la landing
- src/app/shared/data: datasets de ejemplo
- src/app/shared/models: interfaces tipadas
- src/app/shared/directives: animaciones reutilizables
